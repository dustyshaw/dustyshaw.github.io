Conway's Game of Life

I decided to create a visualization of Conway's famous "Game of Life," a popular cellular autonoma algorithm.

There are three rules to the Game of Life:

  1. If a cell has more than three neighbors and is alive, it will die.
  2. If a dead cell has 2 or 3 neighbors, it will come alive.
  3. If a living cell has less than 2 living neighbors, it will die.

What is Clojure?

Clojure is a not-so-popular programming language that was written in order to create documents.

It is a functional programming language, meaning that there are not classes or objects like there are in Object Oriented programming languages like C#, JavaScript, or Python.

There are some benefits to using a Functional Programming language:

  • Things are immutable. Because you don't instantiate anything or any classes, things never change. This can remove a lot of bugs.
